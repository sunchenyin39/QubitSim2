# QubitSim2
# 1.Install
~~~ 
1.git clone https://github.com/sunchenyin39/QubitSim2.git
2.cd QubitSim2
3.python setup.py sdist
4.cd dist
5.pip install QubitSim2-1.0.tar.gz -i https://pypi.tuna.tsinghua.edu.cn/simple
~~~
